# Library

A simple JavaScript library management app that lets you add, display, and manage books dynamically in a
table

## Features

- Add new books using a dialog form
- Display books in a clean, centred table
- Animate buttons on click
- Reset form after adding a new book

## Tech Stack 

- **HTML5** - page structure
- **CSS3** - layout, styling, and animations
- **JavaScript (ES6)** - logic for dialog form handling, and DOM updates
